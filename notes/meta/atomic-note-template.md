# Atomic Note Template

아래 틀을 복사해서 새 원자 노트를 시작한다.

## 파일명 규칙

- 영문 `kebab-case`를 사용한다.
- 기술명과 핵심 행동 또는 개념을 함께 넣는다.
- 예: `git-rebase-basics.md`, `postgres-index-selectivity.md`

## 문서 템플릿

```md
# 제목

## 한 줄 요약

## 언제 쓰나

## 핵심 개념

## 실전 예시

## 주의할 점

## 관련 노트
```

## 작성 체크리스트

- 한 문서에 하나의 개념, 문제, 패턴만 다룬다.
- 새 노트는 허브 문서 1곳 이상에서 링크한다.
- 관련 노트는 최소 2개 이상 연결을 시도한다.
- 공개 저장소 기준에 맞게 민감한 정보는 제거한다.
